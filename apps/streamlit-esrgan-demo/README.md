# ESRGAN Super Resolution

Upload a low-resolution image and get a 4× upscaled result using [Real-ESRGAN](https://github.com/xinntao/Real-ESRGAN) (RRDB architecture).

Model weights are downloaded automatically from [`ai-forever/Real-ESRGAN`](https://huggingface.co/ai-forever/Real-ESRGAN) on first run.
